Palindrome checker

Input any word and check if it's a palindrome or not

How it works?

A palindrome is a word that reads the same backward or forward.
Write any word (can include numbers) and check if it's a palindrome or not.
The program is case sensitive, so it means that AA will return true but Aa not.

Created by IvanR using HTML5, vanilla JS, CSS3 and bootstrap for responsive design
